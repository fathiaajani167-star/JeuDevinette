# Jeu de Devinette (Java)

Projet académique consistant à développer un programme de devinette numérique interactif.

## Fonctionnement
- Le programme génère un entier aléatoire dans l'intervalle [1, 100].
- L'utilisateur saisit ses propositions via la console.
- Le programme guide l'utilisateur en indiquant si le nombre secret est plus grand ou plus petit.
- La partie s'arrête en cas de succès ou après 10 échecs consécutifs.

## Prérequis
- Java JDK 8 ou supérieur.

## Compilation et Exécution
1. Ouvrez un terminal dans le dossier du projet.
2. Compilez le fichier :
   ```bash
   javac Devinette.java
